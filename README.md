# FWeChat

### 这是一个基于Vite+Vue3+Ts搭建的仿微信项目

>克隆该仓库到本地机器上然后执行 `npm install` 安装依赖

运行

```bash
npm run dev
```

打包
```bash
npm run build
```

预览打包后的文件
```bash
npm run serve
```
